
# Библиотека на Django
Простой проект на Django, демонстрирующий основные возможности фреймворка: создание, чтение, обновление и удаление записей (CRUD), работа с моделями, формами, шаблонами и фильтрация данных



## Возможности
* Вывод информации - главная страница со статистикой и последними добавленными книгами
* Добавление данных - формы для создания книг и авторов
* Просмотр деталей - подробная информация о каждой книге
* Редактирование и удаление - полный CRUD функционал
* Поиск и фильтрация - расширенный поиск по нескольким критериям
* Адаптивный дизайн - минималистичный современный интерфейс

## Установка и запуск

1. Клонировать репозиторий
    ```bash
    git clone https://github.com/Julia-Mamsheva/ProjectDjango
    cd projectdjango
2. Создать виртуальное окружение

    ```bash
    # Windows
    python -m venv venv
    venv\Scripts\activate
    
    # Linux/Mac
    python3 -m venv venv
    source venv/bin/activate    
3. Установить зависимости
    ```bash
    pip install -r requirements.txt
4. Применить миграции
    ```bash
    python manage.py makemigrations
    python manage.py migrate
5. Создать суперпользователя (для доступа к админке)
    ```bash
    python manage.py createsuperuser
6. Запустить сервер
    ```bash
    python manage.py runserver
# Функционал по страницам
* Главная страница (`/`)
    * Статистика: общее количество книг, доступных книг, авторов
    * Последние добавленные книги
    * Навигация по разделам
* Список книг (`/books/`)
    * Каталог всех книг с краткой информацией
    * Поиск по названию, автору и ISBN
    * Фильтрация по автору, году издания
    * Фильтр "только доступные"
    * Кнопки для просмотра, редактирования и удаления

* Детальная страница книги (`/books/<id>/`)
    * Полная информация о книге
    * Ссылки на редактирование и удаление

* Добавление/редактирование книги (/books/create/ и /books/<id>/update/)
    * Форма со всеми полями книги
    * Валидация данных
    * Сообщения об успешном сохранении

* Удаление книги (`/books/<id>/delete/`)
    * Подтверждение удаления
    * Защита от случайного удаления

* Список авторов (`/authors/`)
    * Все авторы с информацией
    * Количество книг каждого автора
    * Ссылки на книги автора

* Добавление автора (`/authors/create/`)
    * Форма для добавления нового автора



## Пошаговая инструкция создания и запуска проекта

1. Создайте папку
    ```bash
    mkdir test_project
2. Создайте окружение
    ```bash
    python -m venv test_project 
3. Изменение политики выполнения скриптов
    ```bash
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
4. Использовать относительный путь для активации окружения
    ```bash
    .\test_project\Scripts\Activate.ps1
    ```
    * Проверьте успешность активации.
После выполнения команды вы должны увидеть, что приглашение командной строки изменилось и теперь начинается с `(test_project)` (или то как вы назвали свой проект). Например:`(test_project) PS X:\djangoproject>`

5. Установка django
    ```bash
    pip install django
6. Создание проекта
    ```bash
    django-admin startproject first_project
7. Перейдите в папку для запуска проекта
    ```bash
    cd first_project
8. Команда для запуска вашего проекта
    ```bash
    python manage.py runserver
